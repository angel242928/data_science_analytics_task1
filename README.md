# Business Sales Performance Analytics 📊

## 🏢 Project Overview
This repository contains the data analysis and visualization pipeline completed for *Task 1* of the Data Science & Analytics program at *Future Interns*. The project processes transactional retail sales records (train.csv) to uncover critical revenue trajectories, structural consumer demand, and regional financial performance metrics.

## 🛠️ Tech Stack Used
* *Development Environment:* Visual Studio Code (VS Code)
* *Core Language:* Python 3.13
* *Libraries:* Pandas (Data manipulation), Matplotlib & Seaborn (Statistical visualization)

## 🔑 Key Analytical Insights Discovered
* 📈 *Revenue Trajectory:* Sales metrics show clear recurring end-of-year seasonal surges, indicating the company should increase logistics capacity ahead of Q4.
* 📦 *Product Vitality:* Revenue is highly concentrated in the top product category, highlighting an opportunity to run cross-promotional bundles to lift underperforming inventory.
* 🗺️ *Geographic Performance:* Net profit margins vary significantly by geographic zone, showing exactly which regions yield the highest return on investment.

## 📁 Project Deliverables File Directory
* analysis.ipynb - The interactive Python notebook containing cleaning filters and visual generation layers.
* train.csv - The core transactional retail dataset.
* revenue_trends.png - Visual chart output mapping monthly company growth.
* category_performance.png - Visual chart ranking categorical performance.
